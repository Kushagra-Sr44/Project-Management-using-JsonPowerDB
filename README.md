# Project Management Form using JsonPowerDB

## Description
This project is a web-based Project Management Form developed using HTML, Bootstrap, jQuery, and JsonPowerDB (JPDB).  
It allows users to store, retrieve, and update project details such as Project ID, Project Name, Assigned To, Assignment Date, and Deadline.

The application uses JPDB APIs like PUT, GET_BY_KEY, and UPDATE with real-time form validation and database operations.

## Benefits of using JsonPowerDB
- High-performance and lightweight NoSQL database
- Simple REST API-based integration
- Schema-free data storage
- Fast CRUD operations
- Easy to use for web applications

---

## Scope of Functionalities
- Add new project records
- Retrieve existing records using Project ID
- Update existing project details
- Form validation for all input fields
- Enable/disable form fields based on record status
- Reset form to initial state
